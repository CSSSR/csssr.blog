# [`<ParagraphWithImage />`](http://s.csssr.ru/U31J879TR/20200527123603.jpg)

Компонент параграфа c изображением в левом углу. Изображения для параграфа должны находится в папке `public/components/paragraphWithImage`. Принимает следующие пропсы

## `imageName`

Тип данных: `string`

Дефолтное значение: `undefined`

Название папки с изображениями в `public/components/paragraphWithImage`.

## `alt`

Тип данных: `string`

Дефолтное значение: `undefined`

`alt` текст для изображения.
